# CopyFenBiComment

复制粉笔app用户的评价到剪贴板中（长按评价）

# 下载地址

https://github.com/Xposed-Modules-Repo/github.zerorooot.copyfenbi/releases

# 评论过滤

## 使用
在`/sdcard/Android/`下新建一个名为`dz`文件。

# 注意

当前版本(1.2.1)仅支持粉笔app**6.16.54**。

粉笔app**6.16.37**、**6.16.39.32**请使用1.1
